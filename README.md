# Installation
## 1 - Register repository in local library

```
repositories {
  maven { url 'http://mickey305.github.io/JSONAnalyzer/repository/' }
  ...
}
```

## 2 - Compile library

```
dependencies {
  // newest version
  compile 'com.mickey305:library:+@jar'
  ...
}
```

```
dependencies {
  // target version - e.g. version 0.1.1
  compile 'com.mickey305:library:0.1.1'
  ...
}
```
